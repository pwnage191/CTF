#### Challenge Name
Javascript - Obfuscation 1

#### Goal  
Understand the obfuscated JavaScript and find the hidden flag.   

#### Initial Observation  
The page contains an inline JavaScript where the password is hidden using URL encoding
 
#### Approach  
- Opened the webpage and saw a password input field.
- Inspected the page source and opened the JavaScript file.
- Found a suspicious string stored in a variable that looked encoded.
- Copied the encoded string and analyzed it using CyberChef.
  
#### Solution  
- Decoded result revealed the flag.

#### What I Learned  
- Basic JavaScript obfuscation techniques.
- Using CyberChef to identify and decode encoded data.
