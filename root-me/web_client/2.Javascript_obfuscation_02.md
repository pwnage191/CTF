#### Challenge Name
Javascript - Obfuscation 2

#### Goal  
Find the hidden password in the JavaScript code.

#### Initial Observation  
The page has a JavaScript script where the password is hidden inside a URL-encoded string with multiple layers.

#### Approach  
- Opened the page source and found an encoded string.
- Noticed it used `unescape()` and `String.fromCharCode()`.
- Ran the code step by step in the browser console to decode each layer.

#### Solution
- Each time I ran `unescape()` in the console, it revealed the next layer.
- The final layer was `String.fromCharCode()` which gave the password.

#### What I Learned
- Multi-layer obfuscation — each decode reveals the next layer.
- `%25` trick** — double URL encoding to hide the real `%` characters.
- `String.fromCharCode()` — converts ASCII numbers to characters.
- `unescape()` — URL decodes `%XX` encoded strings.
