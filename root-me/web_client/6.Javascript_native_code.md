#### Challenge Name
Javascript - Native code

#### Vulnerability Type
Client-side authentication with JavaScript obfuscation

#### Walkthrough
- Opened the page and saw a password prompt
- Entered random input and received "fail"
- Page became blank after submission
- Inspected the page source
- Found obfuscated JavaScript using unicode variables and encoded characters
- Deobfuscated the script into readable code
- Identified a hardcoded password comparison in client-side logic
- Extracted the correct password from the code

#### What I Learned
- Obfuscation only hides logic, it does not secure it
- Client-side password checks can always be reversed
- Sensitive data should not be stored in frontend code
